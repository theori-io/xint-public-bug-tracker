# Xint Public Bug Tracker
This read-only repository is set up to make it easy to see publicly reported
bugs found with [Theori's](https://theori.io) [Xint](https://xint.io) in
open-source software.

Bugs may be listed with only minimal details while still under embargo or
waiting for patches from upstream maintainers. 

---

Check out [Xint](https://xint.io) to learn more about Xint or to get in touch
about securing your own software!
